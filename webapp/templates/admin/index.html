Import/Export Tool
Bitte beachten

Gelegentlich kann es nötig sein, die Daten in ein anderes Datenbanksystem zu migrieren. Die Exportfunktion auf dieser Seite exportiert alle in der Datenbank enthaltenen Daten in eine Datei. Mit der Importfunktion kann diese Datei wieder importiert werden. Dabei werden keine existierenden Daten überschrieben.

Diese Funktionen ersetzen kein Backup der Datenbank.

Bitte stelle sicher, dass die Daten vollständig importiert werden und die Applikationsversion zwischen Export und Import nicht verändert wird.
